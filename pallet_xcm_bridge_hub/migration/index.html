Structs§

  • OpenBridgeForLane
    This migration does not modify storage but can be used to open a bridge and link it to the specified LaneId. This is useful when we want to open a bridge and use a custom LaneId instead of the pre-calculated one provided by the fn open_bridge extrinsic. Or perhaps if you want to ensure that your runtime (e.g., for testing) always has an open bridge.
