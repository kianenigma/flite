Expand description

A flite representative of [polkadot_sdk::pallet_aura::Config].

This merely exists because pallet_aura itself doesn’t have #[pallet::config(with_default)]. Although, as we established for FliteTimestamp, even if it would, it would have not been useful for this case.

We drop all most trait bounds, as we test it and we are professional.
