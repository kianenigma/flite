Documentation of the API (host functions) available to contracts.

The Current trait might be the most useful doc to look at. The versioned traits only exist for reference: If trying to find out if a specific version of pallet-contracts contains a certain function.

§Note

This module is not meant to be used by any code. Rather, it is meant to be consumed by humans through rustdoc.
